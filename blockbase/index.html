What it does

A launcher built around practical Minecraft sessions.

BlockBaseMC keeps the important things close: create instances, pick versions and loaders, sign in with Microsoft, import existing packs, browse mods, and stop running Minecraft processes without digging through folders.

Profiles Microsoft login Modrinth CurseForge GitHub updates

Instances

Profiles first

Create isolated Java instances with their own Minecraft version, loader, RAM, JVM args, launch args, folder, and selected account.

Accounts

Microsoft login

Add Microsoft Java accounts, keep account cards clean, show profile details, and use browser login instead of a tiny broken popup.

Content

Mods and packs

Browse Modrinth and CurseForge, install into the selected profile, manage installed mods, and import ZIP or existing Minecraft folders.

Runtime

Running instances

Track live Minecraft processes, label them by account/profile, open folders/logs, and kill a chosen running instance.

Built With

Desktop app stack plus Minecraft services.

Electron

Desktop shell, local file access, process launching, and browser-based Microsoft authentication.

JavaScript

Launcher UI enhancements, state handling, Minecraft launch preparation, imports, and API integrations.

Mojang + Microsoft

Minecraft version metadata, asset/library downloads, Java account auth, and launch session handling.

Modrinth + CurseForge

Project search, modpack workflows, dependency metadata, and content management ideas.
