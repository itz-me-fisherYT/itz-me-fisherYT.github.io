Live Commands

The command menu currently running in Sentra.

Utility

/help

Show the current command panel and what Sentra can do right now.

Utility

/ping

Check whether the bot is online and how responsive it is.

Utility

/server

View quick server stats like member count, age, and owner ID.

Utility

/user

Inspect a user profile, account age, and join date.

Moderation

/ban

Remove a member permanently with an optional reason.

Moderation

/kick

Kick a member while keeping the rest of the server intact.

Moderation

/timeout

Silence a member for values like 10m, 2h, or 1d.

Moderation

/purge

Bulk delete recent messages during cleanup or spam events.

Moderation

/lock

Stop everyone from sending messages in the current channel.

Moderation

/unlock

Open the channel back up once things are under control.

Warnings

/warn

Add a recorded warning to a member with a reason and case ID.

Warnings

/warnings

Review a member's saved warning history directly from Discord.

Hosted Setup

Two deploys, one Sentra experience.

  1. Website: host the public Sentra page on GitHub Pages.
  2. Bot: host the Discord bot from the bot app on LiteByte.
  3. Shared Brand: keep the avatar, command list, and tone aligned.
  4. Future Links: add invite and support server buttons when they are ready.
Sentra protects your server quietly in the background, steps in fast when things go wrong, and keeps moderators in control.
